블록체인 기반 증명서 발급 포털 - 개발

프로젝트 개요

2D 바코드를 이용하여 대학교의 증명서를 PDF로 발급받아 블록체인에 등록하고, 문서의 위변조 여부를 확인할 수 있는 신뢰성 높은 증명서 관리 서비스입니다.

항목내용
기간2019.11 - 2020.03
역할화면 설계, 백엔드 개발, 프론트엔드 개발

백엔드 개발

  • 증명서 블록체인 서버에 연동 개발
  • 2D 바코드 기반 증명서 인증 기능 연동
  • 블록체인 위변조 검증 연동
  • 게시판, FAQ 등 사용자 서비스 화면 전반 관한 백엔드 개발
  • 인도네시아 버전 대응

프론트 엔드 개발

  • 블록체인 및 증명서 관련 화면 개발
