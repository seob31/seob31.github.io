## Project Overview
A centralized software component-based ESB/EAI system that enables application integration within the enterprise. This is an enterprise-grade message integration platform supporting diverse channels for data aggregation and monitoring.

**Period:** 2023.09 - Present  
**Team Size:** 8 members

---

## Key Responsibilities
- **Front-end (React)** screen development and UI/UX enhancement
- **Back-end** development and system optimization
- Channel library patch feature development
- Log management system development
- Interface simulation feature implementation
- Message segmentation and transmission development
- Message monitoring functionality
- Product LMS technical support

---

## Technology Stack
**Backend:** Java, Spring Boot, Spring WebFlux, JPA, Quartz, JWT  
**Frontend:** React, JavaScript  
**Database:** Oracle, JPA  
**Infrastructure:** Docker, AWS, VM ware, Jenkins  
**Tools:** IntelliJ, Eclipse, VS Code, GitHub Copilot, Notion, GitLab, DBeaver  
**Architecture:** Saga Pattern, Microservices

## Major Achievements
1. **SP Certification and GS Certification**

2. **Logging System Optimization**
