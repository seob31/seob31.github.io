## Project Overview
A web-based learning management system designed with 'Education & Game' theme providing entertaining quiz-based training. Developed as an in-house employee education platform.

**Period:** 2018.01 - 2018.02  
**Team Size:** 3 members  
**Role:** Gamified learning system development

---

## Key Responsibilities
- **Screen development** following basic and detailed design specifications
- **Screen design and ERD revision, database normalization**
- User interface design

---

## Technology Stack
**Backend:** Java, Spring Framework  
**Frontend:** HTML, JavaScript, jQuery, Ajax  
**Database:** Oracle, SQL, ER Modeling  
**Tools:** Eclipse  

---

## Core Features
- Quiz-based learning content
- Education & Game integrated environment
- Progress tracking and grade inquiry
- Real-time feedback
- Ranking system

---

## Project Highlights
- Gamification-based education
- Rapid development (approximately 2 months)
- User-friendly interface
- Efficient database design
- Small team collaboration experience
