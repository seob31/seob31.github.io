## Project Overview
A media file leakage prevention service that applies forensic watermarking and DRM (Digital Rights Management) to photos, audio, and video files, providing HLS and Dash-based video playback and download services.

**Period:** 2023.01 - 2023.05  
**Team Size:** 7 members  
**Role:** System design and infrastructure management, CI/CD implementation

---

## Key Responsibilities
- **System, database, and server design**
- AWS management and configuration, VPC setup
- Jenkins and CI/CD configuration, Docker development
- Technical support for domestic and international developers
- Development oversight for main, composer, and forensics 3 projects
- AWS GPU server management and operation

---

## Technology Stack
**Backend:** Java, Spring Boot, Mybatis, Gradle  
**Database:** PostgreSQL, SQL  
**Infrastructure:** AWS (EC2, EFS, S3, VPC, ALB), Docker, Jenkins, GitLab  
**Tools:** Eclipse, VS Code, DBeaver, WinSCP, Putty, ffmpeg  
**DevOps:** CI/CD Pipeline, Docker Container, AWS Scaling
