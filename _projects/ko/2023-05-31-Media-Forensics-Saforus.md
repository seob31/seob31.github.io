## 📋 프로젝트 개요
사진, 오디오, 비디오 파일에 포렌식 워터마킹 및 DRM(디지털 권리 관리)을 적용하여 HLS, Dash 기반 동영상 재생 및 다운로드 서비스를 제공하는 미디어 유출 방지 플랫폼입니다.

| 항목 | 내용 |
|------|------|
| **기간** | 2023.01 - 2023.05 (약 5개월) |
| **팀 규모** | 7명 |
| **역할** | 시스템/인프라 리드, DevOps 담당 |

---

## 🎯 프로젝트 배경
미디어 콘텐츠의 무단 배포 및 유출로 인한 저작권 침해 문제를 해결하기 위해, 고급 포렌식 기술과 DRM 기술을 통합한 안전한 미디어 배포 솔루션을 구축하고자 함.

---

## 💼 담당 역할

### 시스템 아키텍처 설계
- 전체 시스템 설계 및 데이터베이스 스키마 설계
- AWS 기반 클라우드 인프라 설계 (VPC, 보안 그룹, 네트워크 구성)

### AWS 클라우드 관리
- AWS EC2, EFS, S3, ALB 운영 관리
- GPU 서버 관리 및 포렌식 처리 최적화
- 자동 스케일링 전략 수립 및 구현

### CI/CD 파이프라인 구축
- Jenkins 기반 CI/CD 파이프라인 설계 및 구현
- Docker 기반 컨테이너 오케스트레이션
- 배포 자동화 및 스테이징 환경 관리

### 개발팀 지원
- 국내외 개발자 기술 지원 및 멘토링
- 메인, 컴포져, 포렌식 3개 프로젝트 개발 진행

---

## 🔧 주요 기능 및 구현

- **포렌식 워터마킹**: 미디어 파일에 식별 가능한 추적 정보 삽입
- **DRM 적용**: 동작 환경별 콘텐츠 보호 정책 적용
- **HLS/Dash 스트리밍**: 적응형 비트레이트 스트리밍 지원
- **포렌식 분석**: 불법 복제 추적 및 검증 기능
- **GPU 기반 처리**: 고성능 미디어 변환 및 분석

---

## 🛠️ 사용 기술
**Backend:** Java, Spring Boot, Mybatis, Gradle  
**Database:** PostgreSQL, SQL  
**Infrastructure:** AWS (EC2, EFS, S3, VPC, ALB), Docker, Jenkins, GitLab  
**Tools:** Eclipse, VS Code, DBeaver, WinSCP, Putty, ffmpeg  
**DevOps:** CI/CD Pipeline, Docker Container, AWS Scaling

---

## 🏆 주요 성과

1. **포렌식 처리 정확도 향상**
   - ffmpeg 명령어 최적화로 **정확도 20% 향상**
   - 음성/영상 처리 알고리즘 개선 완료

2. **AWS 인프라 자동화**
   - EC2 자동 스케일링 지표 확립
   - 자동 스케일 인/아웃 전략 구현으로 비용 최적화

3. **해외 시연 성공**
   - 긴급 해외 시연서버 구축 주도
   - **US 유니버설 스튜디오에서 성공적인 시연** 진행
