| 항목 | 내용 |
|------|------|
| **기간** | 2018.01 - 2018.02 (약 2개월) |
| **역할** | 화면 개발, 화면 설계, 백엔드 개발 |
| **상태** | 사내에서 기획한 프로젝트, 인력 파견으로 인해 프로젝트 초기에만 개발 투입 |
