## 📋 프로젝트 개요
'교육과 게임'을 결합한 웹 기반 학습 관리 시스템입니다. 퀴즈 형식의 재미있는 연수를 제공하며, 사내 직원 교육용 플랫폼으로 개발되었습니다. 게임화를 통해 학습 참여도와 효과를 극대화합니다.

| 항목 | 내용 |
|------|------|
| **기간** | 2018.01 - 2018.02 (약 2개월) |
| **팀 규모** | 3명 |
| **역할** | 전체 화면 개발, 화면 설계 담당 |

---

## 🎯 프로젝트 배경
직원 교육의 참여도와 효율성을 높이기 위해 게임화 기법을 적용한 학습 시스템이 필요했으며, 즐거운 학습 경험과 체계적인 진도 관리를 목표로 함. 빠른 개발 일정 내에 완성도 높은 플랫폼 구축이 요구됨.

---

## 💼 담당 역할

### 화면 개발
- 기본설계, 상세설계서에 기반한 **각종 화면 개발**
- **사용자 인터페이스 설계**
- 반응형 UI/UX 구현

### 데이터베이스 설계
- **ERD 수정 및 최적화**
- **데이터베이스 정규화**
- 효율적인 스키마 설계

### 풀스택 개발
- Front-end 개발 (HTML, JavaScript, jQuery, Ajax)
- Back-end 로직 구현
- 게임화 요소 구현

---

## 🔧 주요 기능 및 구현

- **퀴즈 기반 학습 콘텐츠**: 다양한 형식의 문제 출제 및 풀이
- **게임화 학습**: 점수, 배지, 랭킹을 통한 동기부여
- **진도 관리**: 학습자의 학습 진도 추적 및 관리
- **성적 조회**: 개인 및 팀 성적 분석 및 리포팅
- **실시간 피드백**: 답변에 대한 즉시 채점 및 해설 제공
- **랭킹 시스템**: 경쟁을 통한 참여도 증대
- **대시보드**: 교육 현황 한눈에 파악

---

## 🛠️ 사용 기술
**Backend:** Java, Spring Framework  
**Frontend:** HTML, JavaScript, jQuery, Ajax  
**Database:** Oracle, SQL  
**Design:** ER Modeling, Database Normalization  
**Tools:** Eclipse

## 🏆 주요 성과

- **신속한 개발 완성**
  - **약 2개월 내 전체 시스템 개발 완료**
  - 효율적인 팀 협업으로 일정 준수

- **게임화 학습 플랫폼 구축**
  - 교육과 게임을 통합한 혁신적 학습 경험
  - 직원 교육의 참여도 및 흥미도 향상

- **효율적인 DB 설계**
  - ERD 설계 및 정규화를 통한 최적화된 데이터 구조
  - 확장 가능한 데이터베이스 아키텍처 구축

---

## ⭐ 프로젝트 특징

- **게임화(Gamification) 교육**: 점수, 배지, 랭킹을 통한 학습 동기부여
- **신속한 개발**: 2개월의 짧은 기간 내 완성
- **사용자 친화적 인터페이스**: 직관적이고 재미있는 UI/UX
- **효율적인 DB 설계**: 정규화된 데이터 모델과 최적화된 쿼리
- **소규모 팀 협업**: 3명의 팀으로 높은 완성도 달성
