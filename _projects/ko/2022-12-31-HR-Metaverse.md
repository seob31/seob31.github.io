## 프로젝트 개요
가상의 3D 공간에서 출근, 근태관리, 회의 등 회사 생활의 모든 업무를 메타버스 환경에서 수행할 수 있는 혁신적인 인사관리 플랫폼.

| 항목 | 내용 |
|------|------|
| **기간** | 2022.08 - 2022.12 |
| **역할** | 백엔드 시스템 설계 및 개발 리드 |
| **상태** | 1차 개발 중 우선순위가 높은 프로젝트 참여로 프로젝트 개발 보류 |

<br>

## 담당 역할

### 시스템 설계 및 백엔드 개발
- 시스템 설계(MSA)
- 아키텍쳐링
- Unity와의 REST API 서버 개발

### 인프라 및 운영
- AWS 환경의 인프라 구축(EC2, ALB, EFS 등)
- git -> Jenkins -> Docker and Local 기반 배포 환경 관리(CI/CD)

### 국제 팀 협력
- 프로젝트에 속한 해외 개발자 서포트 (Front-end 2, Back-end 1, Unity 1)  
  -> 기술 지원, 프로젝트 관련 내용 및 지시사항 전달
- Unity 개발자의 게임 서버 설치, 설정, 트러블슈팅 전반 지원

---

##  사용 기술
- **Backend**: Java, Spring Boot, Spring Cloud (Gateway, Eureka), MyBatis, JWT
- **Database**: PostgreSQL, Redis
- **Infrastructure / DevOps**: AWS (EC2, EFS, VPC), Docker, Jenkins, GitLab CI/CD
- **Monitoring & Tooling**: Zipkin, Portainer, Swagger
- **Architecture**: MSA
- **Client**: Unity
