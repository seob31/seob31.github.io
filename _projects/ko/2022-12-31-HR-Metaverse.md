## 📋 프로젝트 개요
가상의 3D 공간에서 출근, 근태관리, 회의 등 회사 생활의 모든 업무를 메타버스 환경에서 수행할 수 있는 혁신적인 인사관리 플랫폼입니다.

| 항목 | 내용 |
|------|------|
| **기간** | 2022.08 - 2022.12 (약 5개월) |
| **팀 규모** | 7명 |
| **역할** | 백엔드 시스템 설계 및 개발 리드 |

---

## 🎯 프로젝트 배경
원격근무 시대에 분산된 직원들 간의 협업과 회사 문화 유지를 위해, 메타버스 기술을 활용한 새로운 형태의 가상 업무 공간을 구축하고자 함.

---

## 💼 담당 역할

### 시스템 아키텍처 설계
- 전체 시스템 및 서버 아키텍처 설계
- AWS 인프라 설계 (VPC, 네트워크, 보안 그룹 구성)

### 마이크로서비스 개발
- Unity와의 REST API 서버 개발
- Spring Cloud를 활용한 마이크로서비스 구현

### AWS 클라우드 기반 인프라
- AWS EC2, EFS, VPC 설정 및 관리
- Jenkins CI/CD 파이프라인 구축

### 국제 팀 협력
- Unity 해외 개발자 기술 지원
- 국내외 백엔드 개발자 팀 관리 및 멘토링

---

## 🔧 주요 기능 및 구현

- **가상 공간 기반 출근**: 메타버스 환경에서 실시간 출근 처리
- **근태 관리 시스템**: 자동 근태 기록 및 집계
- **가상 회의실**: 다인원 3D 환경에서의 실시간 협업
- **마이크로서비스 아키텍처**: 확장 가능한 백엔드 설계
- **실시간 데이터 동기화**: Unity와 백엔드 간 실시간 동기화

---

## 🛠️ 사용 기술
**Backend:** Java, Spring Boot, Spring Cloud, Mybatis, Gradle, JWT  
**Database:** PostgreSQL, SQL  
**Infrastructure:** AWS (EC2, EFS, VPC), Docker, Jenkins, GitLab  
**Frontend:** Unity, REST API  
**Monitoring:** Zipkin, Portainer, Swagger  
**Tools:** Eclipse, VS Code, DBeaver, WinSCP, Putty

---

## 주요 기능
- 가상 공간 기반 출근 관리
- 실시간 회의실 예약 및 관리
- 메타버스 환경에서의 근태관리
- REST API 기반 Unity 연동
- 마이크로서비스 아키텍처

---

## ⭐ 프로젝트 특징

- **혁신적인 메타버스 기술**: Unity 기반 3D 가상 공간 구현
- **마이크로서비스 아키텍처**: Spring Cloud 기반 확장 가능한 시스템
- **글로벌 팀 협력**: 해외 개발자와의 효율적인 협업
- **실시간 모니터링**: Zipkin 분산 추적으로 시스템 가시성 확보
- **컨테이너 기반 배포**: Docker와 Portainer를 통한 쉬운 관리
