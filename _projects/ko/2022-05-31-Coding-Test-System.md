## 프로젝트 개요
다중 언어 실행 환경과 채점 기능을 제공하는 코딩 테스트 서비스 개발 프로젝트입니다.  
프로젝트 설계를 단독으로 수행하고, 코어 백엔드 개발부터 배포와 서버 운영까지 전담했습니다.

| 항목 | 내용 |
|------|------|
| **기간** | 2022.02 ~ 2022.05 |
| **역할** | 프로젝트 설계, 코어 백엔드 개발, 서버 운영, SRE |

---

## 담당 역할

### 아키텍처 및 코어 개발
- 전체 서비스 구조 설계
- 코드 실행 및 채점 코어 개발
- 다중 언어 실행 환경 구성

### 운영 및 배포
- Docker 기반 온프레미스 배포 구조 설계
- IDC 서버 운영 및 배포 관리
- 설치 및 운영 자동화 지원

---

## 주요 구현 및 성과

- **12개 언어 지원 코딩 테스트 플랫폼 구축**
  - Isolate 샌드박스를 활용해 **12개 언어**를 지원하는 실행 환경을 구현했습니다.
  - 핵심 개발, 배포, 서버 운영을 모두 전담했습니다.

- **온프레미스 설치 시간 단축**
  - Docker 기반 온프레미스 솔루션을 구성해 설치 과정을 단순화했습니다.
  - 설치 시간을 **25% 이상 단축**했습니다.

---

## 사용 기술
**Backend:** Java  
**Database:** MariaDB  
**Infra & Ops:** Docker, IDC 서버, Shell  
**Security:** Isolate Sandbox  
**Tools:** GitLab, Monday

## 프로젝트 특징
- 소규모 팀에서 설계부터 운영까지 폭넓게 맡은 프로젝트
- 코드 실행 보안성과 운영 편의성을 동시에 고려한 서비스 구조
- 온프레미스 배포 경험을 쌓은 제품형 프로젝트
