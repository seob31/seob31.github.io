## 📋 프로젝트 개요
12개의 프로그래밍 언어를 동시 지원하는 온라인 코딩 테스트 웹 서비스이자 구축형 시스템 개발 프로젝트입니다. 다양한 언어의 안전한 코드 실행 환경을 제공하는 통합 플랫폼입니다.

| 항목 | 내용 |
|------|------|
| **기간** | 2022.02 - 2022.05 (약 4개월) |
| **팀 규모** | 3명 |
| **역할** | 시스템 설계 및 핵심 개발 리드 |

---

## 🎯 프로젝트 배경
온라인 코딩 교육 시장의 확대에 따라, 여러 프로그래밍 언어를 지원하는 신뢰할 수 있는 코딩 테스트 플랫폼이 필요했으며, 빠른 설치가 가능한 구축형 솔루션 개발을 목표로 함.

---

## 💼 담당 역할

### 시스템 아키텍처 설계
- 전체 시스템 설계 및 서버 구성
- 다중 언어 지원을 위한 아키텍처

### 멀티 랭귀지 코어 개발
- **12개 프로그래밍 언어** 동시 지원:
  - Java, Python, C++, C#, JavaScript, Ruby, Go, Rust, PHP, TypeScript, Kotlin, Swift
- 각 언어별 컴파일러/인터프리터 통합

### 배포 및 관리
- Docker 기반 컨테이너 구축
- 설치 자동화 및 배포 시스템 구현

---

## 🔧 주요 기능 및 구현

- **12개 언어 지원**: 다양한 프로그래밍 언어에서의 코드 실행
- **보안된 실행 환경**: Isolate를 이용한 샌드박스 환경
- **자동 채점 시스템**: 다양한 테스트 케이스에 대한 자동 검증
- **플러그 앤 플레이**: Docker 기반 빠른 설치 및 배포
- **실시간 피드백**: 코드 실행 결과 즉시 제공

---

## 🛠️ 사용 기술
**Backend:** Java, Spring Boot, Shell Script, Gradle  
**Database:** MariaDB, SQL  
**Infrastructure:** Docker, WinSCP, Putty  
**Security:** Code Isolation (Isolate)  
**Tools:** Eclipse, GitLab, DBeaver, Monday
