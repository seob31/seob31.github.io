## 프로젝트 개요
교육 이력과 자격증 정보를 관리하고 증명서 발급을 지원하는 블록체인 기반 서비스 프로젝트입니다.  
백엔드와 프론트엔드 개발, 서버 운영, 데이터 보관 구조 개선까지 폭넓게 담당했습니다.

| 항목 | 내용 |
|------|------|
| **기간** | 2019.11 ~ 2022.01 |
| **팀 규모** | 8명 |
| **역할** | 프로젝트 설계, 백엔드/프론트엔드 개발, 서버 운영 |

---

## 담당 역할

### 서비스 개발
- 프론트엔드와 백엔드 기능 설계 및 개발
- 증명서 생성 및 발급 기능 구현
- 블록체인 기반 이력 관리 기능 개발

### 운영 및 데이터 관리
- AWS 환경 서버 운영
- 모니터링 도구 도입 및 운영 안정화
- 데이터 분리·보관 자동화 로직 개발

---

## 주요 구현 및 성과

- **증명서 생성 기능 내재화**
  - 증명서 생성 기능을 설계하고 프론트엔드(JavaScript)와 백엔드(Java)를 모두 직접 개발했습니다.
  - 외부 라이브러리 의존성을 제거해 **약 2천만원의 개발 비용을 절감**했습니다.

- **데이터 보관 자동화 및 보안 강화**
  - MariaDB 저장 프로시저와 스케줄 이벤트를 활용해 데이터 분리·보관 자동화 시스템을 구축했습니다.
  - 프로시저 단위 로깅 시스템과 보안 대응을 통해 **ISMS-P 인증 획득에 20% 기여**했습니다.

- **운영 환경 안정화**
  - 모니터링 도구를 도입하고 안정적인 서버 운영에 기여했습니다.

## 사용 기술
**Backend:** Java, MyBatis, Maven  
**Frontend:** HTML, JavaScript, jQuery, Ajax  
**Database:** MariaDB, MySQL, SQL  
**Infrastructure:** AWS, Docker, Shell, HAProxy, Portainer  
**Monitoring:** Prometheus, Grafana  
**Tools:** SVN, Redmine

## 프로젝트 특징
- 백엔드와 프론트엔드를 함께 책임진 풀스택 프로젝트
- 운영 비용 절감과 보안 인증 대응을 동시에 만든 경험
- 데이터 보관, 로깅, 모니터링까지 운영 관점이 반영된 서비스
